### Known Integration Notes

1. **INST-08 divergence:** API key prompts were removed because LXC containers lack a TTY during `lxc-attach` execution. Users edit `.env` post-install. This is a deliberate design decision documented in Phase 1 and Phase 2 research.

2. **UPD-04 implicit preservation:** `.env` is never touched in `update_script()`, but `git checkout` could overwrite it if upstream ever tracks `.env`. Low risk — OpenMontage has `.env` in `.gitignore`.
